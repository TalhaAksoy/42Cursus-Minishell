## Run Locally

Clone the project

```bash
  git clone https://github.com/TalhaAksoy/42Cursus-Minishell.git
```

Go to the project directory

```bash
  cd 42Cursus-Minishell
```

Install dependencies

```bash
  make
```

Run Project

```bash
  ./minishell
```
